# Data-Cleaning-File-Processing-Web-App-Streamlit-
An interactive data cleaning web application using Streamlit that allows users to upload CSV/JSON files and process them in real time. Utilized Pandas to handle missing values, remove duplicates, and standardize inconsistent data. Enabled quick preprocessing and preview of cleaned datasets, reducing manual data handling effort.

Key Points:
nteractive web application using Streamlit to upload and process CSV/JSON files
Implemented data cleaning pipelines using Pandas (handling missing values, duplicates, and inconsistencies)
Automated preprocessing tasks like normalization, formatting, and null value imputation
